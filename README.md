# kinematics
To simplify the computation of kinematics, I open the relative code to advance the robotics industry as my tiny contribution.
为了简化运动学的计算，我公开了相关的代码，为机器人工业的发展贡献绵薄之力。
## usage
According to the DH parameter and the offset of initial pose of your robotic arm,you could use it to compute four, five or six axis robotic arm.
根据DH参数和机械臂的初始姿态偏移量(默认竖直向上)，可以用它来计算四轴、五轴或六轴机械臂。
### detail
#### six axis
no change
#### five axis
change the four axis angle range to 0
#### four axis
change the four axis angle range to 0 and make dt to 0
